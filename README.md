# Three.js 粒子圣诞树 🎄

一个使用 Three.js 实现的粒子圣诞树动画效果。包含圣诞树、星星、满天星等元素，使用着色器实现绚丽的粒子效果。整个圣诞树由粒子系统构成，通过 GLSL 着色器实现炫丽的动画效果。

## 项目运行

1. 安装依赖
   ```bash
   npm install
   ```

2. 开发环境运行
   ```bash
   npm run dev
   ```

3. 打包
   ```bash
   npm run build
   ```

## 技术栈

- Vue 3：前端框架
- Three.js：3D 渲染引擎
- GLSL 着色器：自定义粒子效果
- Vite：构建工具
- lil-gui：交互式调试面板
